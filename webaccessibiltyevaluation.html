Common Issues Found

❌ Error — All Pages
Missing/Invalid Language
The <html> element is missing a valid lang attribute. This should be set (e.g. lang="en") on every page.
⚠️ Contrast — Multiple Pages
Very Low Contrast
Several pages (Home, Spotify, Sales, Vietnam) have many elements with insufficient contrast ratios between foreground and background colors.
⚠️ Alert — Multiple Pages
Missing Alt Text / Suspicious Links
Various alerts indicate possible missing image alt text and redundant or suspicious link text across multiple pages.
✅ Positive — Vietnam Page
Correct Language Tag
The Vietnam page correctly sets lang="vi" on the HTML element, making it the only page with no language error.
